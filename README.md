# m-index-lab

## 简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
仓库内容以静态页面为主，不针对任何特定网站或域名，仅作为页面集合的存储与展示空间。

## 目录说明

```
m-index-lab/
├── index.html          # 仓库主页 / 页面导航
├── pages/              # 存放各独立 HTML 页面
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md           # 本文件
```

- `pages/`：每个 HTML 页面独立存放，命名清晰，便于直接访问。
- `assets/`：页面共用的样式、脚本或图片资源，避免重复引用。

## 页面归档说明

- 每个页面均为独立完整的 HTML 文件，可直接在浏览器中打开。
- 页面内容可能包含演示、实验、笔记或工具，不构成任何正式服务或产品。
- 页面不依赖外部服务器，所有资源尽量内联或置于 `assets/` 目录下。
- 建议在 `pages/` 内根据功能或主题建立子目录，保持结构清晰。

## 使用方式

1. 克隆或下载本仓库。
2. 在本地直接打开 `index.html` 或 `pages/` 下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 部署，将仓库内容发布为静态网站。

## 维护说明

- 本仓库仅用于归档个人或协作编写的 HTML 页面，不保证时效性与兼容性。
- 欢迎提交 Pull Request 增加新页面或改进现有页面，但请确保内容不涉及侵权、隐私或敏感信息。
- 页面内容可能随时更新或移除，恕不另行通知。
